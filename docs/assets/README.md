# GitHub Preview Assets

Các file SVG trong thư mục này là preview nhẹ để README hiển thị đẹp trên GitHub.

Khi cần demo portfolio nghiêm túc hơn, hãy thay hoặc bổ sung ảnh chụp thật:

- `dashboard.png`
- `copilot.png`
- `semantic-mapping.png`
- `metric-builder.png`

Gợi ý chụp ảnh:

1. Chạy `make backend`.
2. Chạy `make frontend`.
3. Upload một dataset demo.
4. Chụp Smart Dashboard, Ask AI, Semantic Mapping và Metric Builder.
