# Raw Datasets

Đặt các dataset raw dùng để chạy demo/eval thủ công trong thư mục này.

Các file raw thường lớn hoặc có license riêng nên không được commit lên GitHub.
Ví dụ:

- `Amazon Sale Report.csv`
- `online_retail_09_10.csv`
- `sample_-_superstore.xls`
- `HR-Employee-Attrition.csv`
- `Marketing+Data/marketing_data.csv`

Nếu cần chia sẻ dataset, hãy ghi nguồn tải trong README hoặc docs thay vì commit file trực tiếp.
